# Agents 索引

本目录包含 eval-agent 系统的所有 Agent。

## Agent 列表

| Agent | 职责 | 输入 | 输出 |
|-------|------|------|------|
| [InputAgent](./InputAgent.md) | 解析用户输入 | 用户输入 (URL/路径) | 项目信息 |
| [LoaderAgent](./LoaderAgent.md) | 加载/克隆项目 | 项目信息 | 项目路径 |
| [CICDAgent](./CICDAgent.md) | CI/CD 分析 | 项目路径 | 分析报告 |
| [ReviewerAgent](./ReviewerAgent.md) | 报告验证 | LLM 响应/报告 | 验证结果 |
| [ReporterAgent](./ReporterAgent.md) | 生成 HTML | 分析报告 | HTML 报告 |
| [CompareAgent](./CompareAgent.md) | 项目对比 | 两个项目 | 对比报告 |
| [IntentParserAgent](./IntentParserAgent.md) | 意图解析 | 自然语言 | 结构化命令 |

## 执行流程

### 分析流程

```
用户输入
    │
    ├── 传统命令: /analyze path
    └── 自然语言: "帮我分析这个项目"
            │
            ▼
    IntentParserAgent (解析意图)
            │
            ▼
    InputAgent (解析输入)
            │
            ▼
    LoaderAgent (加载项目)
            │
            ▼
    CICDAgent (CI/CD 分析)
            │
            ├── 提取 CI/CD 数据
            ├── 分割 + 并发 LLM 调用
            └── 生成 Markdown 报告
            │
            ▼
    ReviewerAgent (验证报告)
            │
            ├── 验证工作流完整性
            ├── 验证统计数据准确性
            └── 验证概述准确性
            │
            ├── 通过 → ReporterAgent
            └── 不通过 → CICDAgent (重试)
            │
            ▼
    ReporterAgent (生成 HTML)
            │
            ▼
    HTML 报告
```

### 对比流程

```
用户输入
    │
    ├── 传统命令: /compare project_a project_b
    └── 自然语言: "对比这两个项目"
            │
            ▼
    IntentParserAgent (解析意图)
            │
            ▼
    CompareAgent (对比分析)
            │
            ├── 加载两个项目数据
            ├── 计算维度得分
            ├── LLM 分析架构差异
            └── 生成对比报告
            │
            ▼
    HTML 对比报告
```

## Agent 接口规范

### 标准 Agent 接口

所有 Agent 应遵循以下接口规范：

```python
class BaseAgent:
    def __init__(self, llm: Optional[LLMClient] = None):
        """初始化 Agent"""
        self.llm = llm
    
    def run(self, state: dict) -> dict:
        """
        执行 Agent
        
        Args:
            state: 输入状态字典
            
        Returns:
            更新后的状态字典
        """
        raise NotImplementedError
```

### 状态字段规范

| 字段 | 类型 | 描述 |
|------|------|------|
| `current_step` | `str` | 当前步骤标识 |
| `errors` | `list[str]` | 错误列表 |
| `warnings` | `list[str]` | 警告列表 |

## LangSmith 可观测性

所有 Agent 在 LangSmith 中可被追踪，包括：

- Agent 入口和出口
- LLM 调用
- 工具调用
- 执行时间和 Token 消耗

### 追踪示例

```python
from langsmith import traceable

@traceable(name="CICDAgent", run_type="agent")
def run(self, state):
    # LangSmith 自动追踪
    ...
```

## 添加新 Agent

1. 在 `agents/` 目录创建新的 Agent 文件
2. 创建对应的 `AgentName.md` 文档
3. 在 `agents/__init__.py` 中导出
4. 更新本索引文档
